## 主題
NixOS 上配置 Docker 支持 GPU 加速

## 核心結論
- 启用 Docker 的 NVIDIA GPU 支持需要特定配置
- 注意 cgroup 版本 compatibility 问题
- rootless Docker 配置需额外注意 GPU 访问权限

## 關鍵步驟
1. 在 configuration.nix 添加以下配置：
   ```nix
   virtualisation.docker.enableNvidia = true;
   ```
2. 安裝 NVIDIA 驱動：
   ```nix
   hardware.nvidia.package = config.boot.kernelPackages.nvidiaPackages.stable;
   ```
3. 安裝 nvidia-container-toolkit：
   ```nix
   environment.systemPackages = with pkgs; [nvidia-container-toolkit];
   ```
4. 重建系統並生效配置：
   ```bash
   sudo nixos-rebuild switch
   ```
5. 验證 GPU 支持：
   ```bash
   docker run --gpus all nvidia/cuda:11.0-base nvidia-smi
   ```

## 適用場景
- 在 NixOS 系統上需要使用 Docker 進行 GPU 加速的場景，如深度學習、科學計算等。
