# 2026-03-22 系統重裝後全面修復報告

## 關鍵踩坑記錄

### Nix Flake 純求值限制
- `builtins.readFile "/etc/xray/config.json"` 在 Flake 模式下失敗
- 解法：用 `builtins.toJSON` + `pkgs.writeText` 內聯配置

### node_modules 被 git 追蹤
- 1750 個文件被意外追蹤
- 解法：`.gitignore` 加 `node_modules/` + `git rm -r --cached`

### GNOME 覆蓋輸入法環境變數
- GNOME 會把 `GTK_IM_MODULE` 等設為 `ibus`
- 解法：`extraGSettingsOverrides` 禁用 + `excludePackages`

### Firefox SOCKS 超時
- xray 只提供 HTTP 代理在 7890，Firefox 用 SOCKS 協議連接 → 協議不匹配
- 解法：`user.js` 明確設定 HTTP 代理 + 新增 SOCKS5 在 7891
